# Student Grade Analyzer (Python CLI)

A command-line application built using Python to store and analyze student grades.

##  Features
- Add student marks subject-wise
- Calculate average marks
- Identify highest and lowest scoring subjects
- View all stored students
- Persistent storage using JSON

##  Technologies Used
- Python
- JSON File Handling
- Functions & Dictionaries

##  How to Run

1. Clone the repository:
   git clone https://github.com/yourusername/student-grade-analyzer.git

2. Navigate into the folder:
   cd student-grade-analyzer

3. Run the program:
   python grade_analyzer.py
